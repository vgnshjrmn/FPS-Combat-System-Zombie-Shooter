# FPS Combat System – Zombie Shooter Demo

A Unity-based FPS combat system demonstrating player movement,
weapon shooting, enemy AI behavior, and in-game UI systems.

## Features
- Custom FPS Controller (movement, camera look, jump)
- Raycast-based shooting and weapon handling
- Zombie enemy AI (chase, attack, damage)
- Health and damage system
- Pause & Resume functionality
- Settings UI with music and SFX volume control

## Systems Overview
Player Controller → Weapon System → Raycast Shooting → Enemy AI → Damage System  
UI Manager → Pause Menu → Audio Settings (Audio Mixer)
